	Sockets: is a communication link btw two machines or programs, it allows sending and reciving 
data using protocols such as TCP & UDP.

	there is two types of sockets , Stream sockets which use TCP and Datagram sockets using UDP.

	Key metodes:
- Socket: Create a new communication
- Bind: Attach a local address to a socket
- Listen: Announce willingness to accept connections
- Accept: Block caller until a connection request arrives
- Connect: Actively attempt to establish a connection
- Send: Send some data over a connection
- Receive: Receive some data over a connection
- Close: Release the connection
